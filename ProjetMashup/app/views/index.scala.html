Welcome to the Mashup course @ UNIGE

Covid-19 API Project

Données mondiales: Nombre de cas confirmés et nombre de vaccins administrés dans le monde

Toutes les requêtes sont envoyés à l'adresse suivante: http:/localhost:9000

Requête pour les données sur tous les pays du monde: http:/localhost:9000/countries

Requête pour les données sur un pays en particulier: http:/localhost:9000/countries/*Nom du pays*

Carte de données concernant les décès dues au Covid-19 dans le monde: http:/localhost:9000/deaths

Carte de données concernant la vaccination contre le Covid-19 dans le monde: http:/localhost:9000/vaccines
